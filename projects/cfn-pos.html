POS & Inventory System

CFN Ice Cream Parlour

An offline Point of Sale (POS) and inventory management system built for small to medium ice cream parlours. Optimized for real-world shop environments with billing, stock tracking, and reporting.

Project Details

Python 3 Tkinter (GUI) SQLite ESC/POS Printing ReportLab
Download EXE View Source

Project Overview

CFN Ice Cream Parlour POS System is designed to run fully offline on Windows-based POS setups. It supports table-based billing, real-time stock deduction, kitchen order tickets (KOT), and thermal printer integration, making it suitable for daily operations in busy ice cream shops. The system stores all data locally using SQLite for maximum reliability.

Key Features

✔
Billing & POS: Table-based billing, dynamic item updates, and thermal printer support.
✔
Inventory Control: Automatic stock deduction on billing and low-stock alerts.
✔
KOT Management: Send orders directly to the kitchen with thermal slip printing.
✔
Reporting: Daily sales analytics and bill history with CSV/PDF exports.
✔
Admin Panel: Secure management of items, categories, and bulk CSV imports.

What I Learned

Built a complete offline POS system handling real shop workflows, including billing logic, inventory synchronization, kitchen order processing, and printer integration. Gained hands-on experience with desktop GUI development, local databases, ESC/POS thermal printing, and designing reliable software for offline environments.
