Inquiry-Based E-Commerce

KK Enterprise

An inquiry-based e-commerce platform built for a fireworks business, allowing customers to browse products, add items to an inquiry cart, and submit bulk requests without online payments.

Project Details

HTML5 CSS3 JavaScript PHP (PDO) MySQL Bootstrap
Live Demo

Project Overview

This system enables customers to explore a wide range of fireworks, filter and sort products, and submit a consolidated inquiry for bulk orders. Instead of direct checkout, the platform focuses on lead generation and customer communication, which is more suitable for local and wholesale businesses.

Key Features

✔
Responsive UI: Optimized for all devices for a seamless browsing experience.
✔
Inquiry Cart: Consolidated request system for bulk product inquiries.
✔
WhatsApp Integration: Direct communication link for quick customer support.
✔
Secure Admin Panel: Full control over products, categories, and inquiry tracking.
✔
Bulk Import: Effortlessly add large numbers of products via CSV files.

What I Learned

Developed a complete inquiry-based e-commerce workflow, implemented admin-side CRUD operations, handled bulk data import, and built a system tailored to real business needs instead of a standard payment-driven model.
